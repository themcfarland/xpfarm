How It Works

1. Pick a data source — choose what to query (Targets, Web Assets, Ports, Vulns, CVEs).
2. Choose output columns — click chips to toggle fields. Unique only deduplicates results.
3. Add conditions — every filter value is a regex pattern (case-insensitive). Use the NOT toggle to exclude matches.
4. Search — results show exactly the columns you selected.

Regex Examples:
admin — matches anything containing "admin"
^(critical|high)$ — exact match: "critical" or "high"
^80$ — exact port 80
\.js$ — URLs ending in .js
^https?:// — URLs starting with http/https
CVE-202[3-5] — CVEs from 2023–2025
react|vue|angular — any of these tech stacks
.* — match everything (show all)
